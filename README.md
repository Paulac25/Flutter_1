# Flutter
Flutter es un framework de desarrollo de aplicaciones móviles creado por Google
que permite construir aplicaciones nativas para Android, iOS, y más plataformas desde una
única base de código. Uno de sus puntos fuertes es su motor gráfico rápido, basado en Dart,
que permite que las interfaces de usuario sean fluidas y altamente personalizables. Flutter
ofrece un amplio conjunto de widgets que permiten crear interfaces visuales atractivas,
además de ser fácil de aprender, incluso para quienes no tienen experiencia previa en
desarrollo móvil.
